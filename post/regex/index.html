Regex - Some Notes

Posted on Sep 6, 2015
#Regex

Notes on regular expressions. Atom is a great test bed, the search function allows regex!

Control characters
\A	Start of string

\z	End of string

\b word boundary

\d numeric characters, \D not numerics (i.e.. ^\d)

\w word, \W not words

\s space, \S not whitespace

\n newline character

\ escape character.

? 0 or 1 times

+ 1 or more times

^ & $ ^	Start of line, $	End of line

^ negate

* 0 or more times

.	Any single character
Grouping, Ranges and Interval Expressions
() Group characters. For example:
                      (a|b)	a or b

?: non-capturing group

[] Range. For example:
          [abc]	A single character of: a, b, or c
          [a-z]	Any single character in the range a-z
          [a-zA-Z]	Any single character in the range a-z or A-Z          

{} interval expression, {x} matches at least x times, {x,y} matches at least x.  times and upto y times. For example:
                          a{3}	Exactly 3 of a
                          a{3,}	3 or more of a
                          a{3,6}	Between 3 and 6 of a
Modifiers

i ignore case
m multiline modifier
g global modifier
